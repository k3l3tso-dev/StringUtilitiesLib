# String Utilities Library

A C# class library demonstrating unit testing with MSTest.

## Features
- `Reverse(string input)`: Reverses a given string.
- `IsPalindrome(string input)`: Checks if a string is a palindrome.
- `CountVowels(string input)`: Counts the number of vowels in a string.
- `SafeTruncate(string input, int maxLength)`: Truncates a long string and adds an ellipsis.

## Getting Started
1. Clone this repository.
2. Open the `StringUtilitiesLib.sln` file in Visual Studio.
3. Build the solution.
4. Run the tests from Test Explorer.

## Testing
This project includes a full suite of unit tests to validate all functionality and edge cases.
